# Research Summary

The research note found that public physical topology exists at macro and
bookcase granularity, but not at the exact granularity required for a promoted
topological decoding hypothesis.

Accepted as public topology:

- Hellgate Library is the central public corpus and the Fandom page lists the
  books in an overview/bookcase order.
- Isle of the Kings Library has strong shelf anchors: bookcase 21 for *Beware
  of the Bonelords* and bookcase 39 for `6512889672 (Book)`.
- Kharos/Ferumbras Tower is a useful external watchlist source through
  TibiaSecrets, but lacks independent shelf/tile granularity in the checked
  public sources.

Still blocked:

```text
book_id -> exact tile -> shelf side -> slot -> orientation -> map version
```
