## Purpose

This gate tests whether the residual exact `type:length` control stream can be coded or generated by a target-free stateful program trained on prefix books. The decoder is granted book id, book length, previous emitted control state, and remaining length. It is not granted target text, row0 origin, plaintext, copy source, or the true skeleton.

## Decision

No stateful control program is promoted. The exact-action codec is worse than independent type+length declaration and worse than shuffled-control p95; any beam exact hits are therefore not usable generation evidence.

`row0` remains exogenous. Translation/plaintext remain unchanged. The `compression_bound` is unchanged because this gate only tests residual control-stream generation.
