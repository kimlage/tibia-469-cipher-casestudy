# `docs/plans/` — historical iteration plans (superseded)

These are the **raw per-iteration working notes** from the decode effort
(`iter159` → `iter791`). They are process material, retained for **provenance
only**. They are superseded by the canonical
[final report](../469_final_report.md) and summarized, with verdicts, in
[wiki page 6 — Attempts & Dead Ends](../wiki/06-attempts-and-dead-ends.md).

> ⚠️ Some figures in these files predate the 2026-06-09 audit corrections (e.g.
> the χ² values 4292/4656/9817 were recomputed to 4296/4850/9408). **Trust the
> [final report](../469_final_report.md) wherever they differ.**
