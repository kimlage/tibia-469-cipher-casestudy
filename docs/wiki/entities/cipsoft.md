# CipSoft GmbH

**CipSoft GmbH** is the German studio that develops and operates *Tibia*, the
MMORPG in which the Bonelord "469" cipher appears. CipSoft authored the source
material this project studies: the 70 Bonelord books, the NPC phrases, and the
official poll/event strings.

## Why CipSoft is the pivotal entity

The project's verdict is that **only CipSoft-attested ground truth could overturn
it** ([open questions](../09-open-questions.md)). Everything the community has —
including this project — works from the same digit strings with no key. The one
category of evidence that would move the needle is something only CipSoft can
provide:

- an official book → plaintext pair, or
- an official symbol/meaning table for the 14 book symbols, or
- a new officially-glossed phrase long enough to test the book map.

If such material ever appears, the [`row0` mechanical model](../02-data-and-method.md)
is the validated substrate to test it against.
